📌 Overview

This project focused on capturing and analyzing network traffic using Wireshark to understand how data packets are transmitted between systems during web communication. The analysis involved monitoring packet transfer while accessing the Honeypot Cloud
 website through Google Chrome and observing real-time network behavior.

The project provided practical exposure to packet analysis, TCP handshake mechanisms, UDP communication behavior, and protocol-level traffic inspection used in networking and cybersecurity analysis.

🛠 Tools & Technologies
Wireshark
Google Chrome
TCP/IP Networking
TCP & UDP Protocols
Packet Capture & Analysis
Virtual / Local Network Environment

⚙️ Tasks Performed
Captured live network traffic using Wireshark
Monitored packet transfer between source and destination IPs
Accessed the Honeypot Cloud website through Chrome for traffic generation
Analyzed TCP three-way handshake packets
Observed UDP traffic behavior and packet transfer
Inspected source IP, destination IP, ports, and protocols
Compared connection-oriented and connectionless communication
Identified packet flow and communication patterns

🔍 Key Learning Outcomes
Gained hands-on experience in packet analysis
Learned how TCP establishes reliable connections
Understood UDP communication behavior and packet loss
Improved understanding of network protocols and packet structures
Developed practical traffic monitoring and analysis skills
Strengthened foundational networking and cybersecurity knowledge

💡 Why This Project?

Network traffic analysis is essential for cybersecurity, troubleshooting, and threat detection. This project provided practical exposure to how data travels across networks and helped build a deeper understanding of communication protocols and packet-level analysis used in real-world environments.
